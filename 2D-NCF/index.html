项目概览

项目简介

《九章幻境》是一款以 Unity 2D 引擎开发，围绕《九章算术》主题打造的探索解谜类游戏。玩家将在幻境中探索未知，发现谜题并解开其中深藏的奥秘。

游戏以 “梦境异世界” 为核心背景，玩家将在梦中进入一个充满奇幻色彩的场景，接受异世界角色的求助，通过与 NPC 交互答题、规避危险怪物、收集回血道具等方式，完成九道题目挑战即可通关，在探索过程中潜移默化地感受传统数学文化的魅力，兼具娱乐性与知识性。

游戏玩法

玩家从游戏开始界面进入后，将置身于初始场景，核心目标是答对九道与《九章算术》相关的题目。流程遵循 “探索场景→寻找 NPC→触发对话→答题→继续探索” 的循环，期间需规避危险、管理血量，直至完成所有题目或血量耗尽结束游戏。

目标受众

10-16 岁青少年群体，该阶段用户处于数学知识学习关键期，游戏的答题玩法可与课堂知识形成联动，帮助其在娱乐中巩固知识，同时培养对传统数学文化的兴趣。

项目愿景

以《九章算术》这一中国古代数学经典为核心，打破传统文化传播的 “书本化” 局限，通过游戏场景、NPC 对话、题目设计等环节，让玩家在探索中了解《九章算术》的历史背景与数学思想，实现 “寓教于乐” 的文化传承目标。

创作过程

阶段一

设计思路

项目初始的概念设计与规划阶段

基于《九章算术》这个核心元素，以 “轻量化交互体验” 作为核心设计理念，本项目通过模块化拆分角色控制、任务系统、UI 反馈等功能，实现低耦合的代码架构；以直观的操作逻辑和清晰的状态反馈降低用户门槛，同时通过 AI 行为的简洁化设计与场景流程的自动化管理，平衡游戏趣味性与开发效率。

“角色探索 + 问答任务” 作为核心玩法，可规划玩家核心行为链：通过移动探索场景→与 NPC 交互接取任务→触发问答挑战→完成任务（累计分数）或失败（生命值耗尽）→场景跳转。围绕该链条拆分核心模块，明确角色控制、任务载体、问答交互、场景流转四大核心组件的职责边界，确保流程闭环。

阶段二

核心功能开发

基于设计思路，实现项目的核心功能与技术框架

1 玩家控制器实现

设计并开发PlayerController核心脚本，实现玩家移动、角色状态管理、交互逻辑及动画状态同步。

关键功能：键盘输入响应（上下左右移动、T 键对话、Q 键触发问题、Esc 键退出）、生命值动态更新与 UI 同步、无敌状态计时器管理。

2 敌人与动物 AI 行为

分别开发EnemyController与AnimalsController脚本，实现自动巡逻逻辑：基于计时器定时切换移动方向（水平 / 垂直可配置），通过Rigidbody2D.MovePosition实现平滑移动；

敌人添加碰撞检测逻辑，与玩家碰撞时触发玩家生命值减少；

动物角色增加动画状态控制。

3 NPC 对话系统

开发NPCDialog脚本，实现 NPC 与玩家的交互功能：对话框自动显示与定时关闭；

绑定 UI 文本组件，支持对话内容动态更新，关联任务状态标记。

4 道具系统实现

开发HealthCollectible脚本，实现生命值道具功能：基于触发检测判断玩家是否拾取；

仅在玩家未满血时生效，拾取后销毁道具并增加生命值，关联音效组件。

阶段三

技术难点与解决方案

开发过程中遇到的挑战及应对策略

问答任务系统

设计Question与QuestionDialog脚本，实现多题问答交互：管理 9 个问题的显示 / 隐藏状态，正确回答时销毁问题触发物并累加分数；

分数达到目标（9 分）时自动跳转胜利场景，错误回答触发玩家生命值扣除；

通过 UI 文本实时显示当前分数，实现任务进度可视化。

最终成果

文化与玩法深度绑定

区别于 “答题 + 场景” 简单叠加的传统教育游戏，本项目将《九章算术》的文化内核与游戏玩法、场景、角色深度绑定，实现 “玩即学、学即玩” 的沉浸式文化传播

低门槛高容错的 “探索 + 生存” 双循环玩法

针对核心受众，设计 “探索找题 + 生存避险” 的双循环玩法，既保证探索的自由度与趣味性，又通过低门槛、高容错的规则降低上手难度，让不同操作水平的玩家都能获得良好体验

流畅的角色动画

通过 Unity Blend Tree技术，实现角色 “上 / 下 / 左 / 右” 四个方向的 “静止 - 移动” 动画无缝过渡，避免传统 2D 游戏的 “卡顿感”
